Steps to install the Steam Data Suite package

1. Import the "SteamDataSuite.unitypackage" into your Unity project
2. Setup your configuration file in the "SteamDataSuiteConfig.asset"
3. There are two ways to manually run the SteamDataSuite
4. Call SteamDataSuite.Initialize() from anywhere in your code, this will look up the config file in the Resources folder to initialize the SDA plugin.
5. Done!

For usage example: See the SteamDataSuiteSampleScene.scene file and the ExampleSteamDataSuite.cs
